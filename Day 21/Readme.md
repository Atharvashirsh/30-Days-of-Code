# Bob and Digits
Time Limit: 2 sec<br>
Memory Limit: 128000 kB
## Problem Statement
Bob loves to play with numbers. He has an integer N. Find the number of digits that N has in base K.
## Input
The input contains two space-separated integers in the following format:
> N K

## Constraints
All values in the input are integers.
> 1 ≤ N ≤ 10<sup>9</sup> <br>
> 2 ≤ K ≤ 10
## Output
Print the number of digits that N has in base K.
## Example
### Sample Input 1
> 11 2
### Sample Output 1
> 4

### Sample Input 2
> 1010101 10
### Sample Output 2
> 7

### Sample Input 3
> 314159265 3
### Sample Output 3
> 18
